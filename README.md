Artificial Intelligence Lab Experiments

This repository contains a collection of Artificial Intelligence Lab programs implemented using Python, Prolog, and HTML.
These experiments cover fundamental AI concepts such as search algorithms, knowledge representation, machine learning, and logic programming.

Objectives

The purpose of this repository is to:

Understand core AI algorithms and techniques

Implement problem-solving methods

Learn Prolog for knowledge representation

Practice basic machine learning concepts

Demonstrate practical AI experiments
